На этой странице собраны ежемесячные бюллетени по безопасности Pixel  и Nexus. Они выпускаются как приложение к бюллетеням по безопасности Android. В них описаны дополнительные обновления системы безопасности и улучшения функциональных возможностей для поддерживаемых устройств Pixel и Nexus.

Уведомления

Устройства Pixel и Nexus начинают получать автоматические обновления системы безопасности (OTA) со дня выхода ежемесячного бюллетеня. Обычно на устройства Nexus обновления приходят в течение 10-11 дней. Образы прошивок Nexus также выходят каждый месяц и публикуются на сайте для разработчиков.

Источники

Перечисленные здесь обновления получены из разных источников: из Android Open Source Project (AOSP), из сообщества Linux и от производителей процессоров. Ссылки на исправления для платформы Android в хранилище AOSP добавляются в бюллетень в течение 24–48 часов после его публикации.
