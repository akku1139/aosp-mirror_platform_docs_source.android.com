New and updated documentation for Android {{ androidQVersionNumber }} CTS features is as follows.

Feature Description Added/Updated
Running CTS tests CTS-Instant test suite is part of the CTS test suite. Updated
CTS Verifier Pro Audio Test The CTS Verifier Pro Audio test verifies the compliance of audio peripherals by measuring round-trip latency through the audio path. Added
Test Harness Mode Test Harness Mode allows the developers to automate a device or a fleet of devices. Added
CTS Verifier MIDI Tests The CTS Verifier MIDI test tests MIDI functionality with USB MIDI interfaces, Bluetooth MIDI interfaces, and a virtual MIDI device path. Added
CTS Shim APEX Developers can preinstall the CtsShimApex package on a device to write CTS tests for APEX management. Updated
CTS Verifier for Instant Apps CTS Verifier for Instant Apps increases Android Compatibility test coverage for Instant Apps by using CTS Verifier to test features that can't be fully automated due to OEM-specific UIs, such as system UIs. Added
Using CTS Verifier CTS Verifier provides tests for APIs and functions that can't be tested on a stationary device without manual input, such as audio quality, touchscreen, accelerometer, and camera. Updated
