The ExtServices module updates framework components such as storage, autofill, notification assistant, and other services that run continually. Together with DocumentsUI and PermissionController, this module ensures that end users get a consistent permissions UI while enabling iteration in response to the changing privacy landscape and government regulations.

Changes in Android {{ androidQVersionNumber }}

The com.android.ext.services APK contains the following services:

  • AutofillFieldClassificationService
  • NotificationAssistantService
  • ResolverRankerService
  • CacheQuotaService

Format and dependencies

The ExtServices module is delivered as an APK file.
