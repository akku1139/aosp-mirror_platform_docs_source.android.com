O painel de integração contínua (ci.android.com, links em inglês) fornece visibilidade ao sistema usado pelo Android Open Source Project (AOSP).

Os colaboradores do AOSP podem usar esse painel para monitorar quando os envios deles são integrados à árvore. A cor do status mostra se a alteração integrada foi criada com êxito em todos os nossos tipos de versão. Para facilitar, os artefatos de cada versão estão disponíveis para download.

Cada linha representa uma versão concluída nos criadores automatizados do Android. Cada coluna representa uma combinação de destino/tipo de versão. Dentro da grade, as versões em andamento e concluídas são mostradas com estas cores de plano de fundo:

  • Amarelo: em andamento
  • Vermelho: com falha
  • Verde: bem-sucedida

Quando uma versão é concluída, é possível fazer o download dos artefatos de versão clicando no ícone get_app(Exibir artefatos), que leva a uma página em que os artefatos podem ser transferidos por download. Clicar em um quadrado abre um painel na parte inferior da tela com uma guia "Detalhes", onde os registros são mantidos, uma guia "Alterações", que lista quais alterações foram incluídas em uma versão, e um link para os artefatos de versão. O painel é atualizado automaticamente conforme novas versões são concluídas.

O painel é encontrado em ci.android.com

Alguns dos atributos do painel são:

  • Nome do branch: nome do branch do git onde a versão ocorre.
  • Destino da versão: Configurador do dispositivo.
  • Tipo de versão: configuração exata do destino, que pode ser user, userdebug ou eng.
    Para mais detalhes, consulte Escolher um destino.
  • Código da versão: código exclusivo de cada versão.
  • Link permanente: link permanente para a página dessa versão em ci.android.com.
  • Link de alterações: link para as alterações incluídas nessa versão.
  • Artefatos da versão: link para ver e fazer o download dos artefatos dessa versão.
